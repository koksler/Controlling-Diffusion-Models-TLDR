# IP-Adapter

Как взять любую картинку и использовать ее как источник вдохновения или референс стиля на лету, без предварительного обучения?

IP-Adapter (Image Prompt Adapter) позволяет использовать изображение в качестве полноценного промпта, наравне с текстом, извлекая из референсного изображения его стиль, композицию, палитру, настроение, концепцию. Что-то похожее на мудборд для настоящего художника.

Адаптированные визуальные эмбеддинги из исходного изображения подаются в те же самые слои Cross-Attention в U-Net, куда подаются и текстовые эмбеддинги. Поэтому промпта теперь два - визуальный и текстовый.
